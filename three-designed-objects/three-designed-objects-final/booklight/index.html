Book light

I like to read and this makes it easier!

  • a. Led

    It uses a led light. (I’m far from an electrician but I know these things consume little energy)

  • b. Clip

    The clip is a great design choice – you can attach it to any book you like. Boom!

  • c. Flexible stem

    Good design because the light can be adjusted easily with this stem.

  • d. Charging port

    Fortunately it doesn’t run on batteries. You can use a typical mini usb cable to charge it up.

  • e. Choice of colors

    You’ve got a warm light which is orange-y and a cold one which is white, currently displayed.
